For IFrame video to work, you cannot simply paste the youtube video link.

You need to use the embed link provided by YouTube.

Here is an example of how to embed a YouTube video:

Click on share, then embed. Youtube will generate the link for you. Copy the embed code from YouTube and paste it into your HTML file.

Make sure to set the width and height attributes for the iframe.

Also, ensure that the allowfullscreen attribute is included to enable full-screen mode.
